Welcome to your very own log store that you can create in Splunk for all your log files using this application. The objective is for you to create your own private index where you can flow your log files to and share it only with those you want to. We explain the workflow required to achieve this and the tools are available with the navigational links below.

The workflow consists of the steps enumerated below. You should be able to :

  1. create your own index List (using this application)
  2. create an LDAP security group (by raising a ticket with your ServiceDesk)
  3. create a role List (using this application)
  4. add the index to this role (using this application)
  5. map an LDAP group to a role (using Splunk)
